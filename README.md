# Extras

## Config

Add to beginning of `printer.cfg`

```
[heaters_extra]
[pid_calibrate_extra]
```

## Changes

### heaters_extra

`heater` option:

`control: pid_v`

### pid_calibrate_extra

gcode command

`PID_CALIBRATE_EXTRA`
